# list-filter
A simple JavaScript DOM project built while learning web development.

## What it does
- Type in the search box to filter the list in real time
- Items that match stay visible
- Items that don't match get hidden

## Concepts used
- querySelector
- addEventListener
- forEach loop
- classList (add / remove)
- toLowerCase & includes

## Tools & Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
